# DESIGN_SYSTEM.md

# Project

Faisal Dev Hub Portfolio (Headless WordPress + Next.js)

---

# Design Vision

This portfolio should not look like a generic developer template.

It should feel like:

* Premium
* Modern
* Confident
* Agency-level
* Luxury dark aesthetic
* Minimal but powerful

The visual quality should feel closer to:

* Apple
* Linear
* Framer
* Webflow Enterprise
* Premium SaaS landing pages

The goal is:

"Make visitors immediately believe they are looking at an experienced professional developer who works with serious businesses."

---

# Brand Personality

The website should communicate:

* Professional
* Trustworthy
* Modern
* Technical
* Clean
* High-end

Avoid:

* Bright colorful gradients
* Cartoon style visuals
* Startup clichés
* Cheap template appearance
* Generic Tailwind look

# Section Spacing

Between Sections:

Desktop:
140px–180px

Tablet:
120px

Mobile:
80px

Never stack sections too tightly.

---

# Border Radius

Cards:
20px

Buttons:
12px

Images:
24px

Large Containers:
32px

---

# Shadows

Avoid heavy shadows.

Use subtle elevation only.

Example:

0 10px 40px rgba(0,0,0,0.25)

---

# Animation Philosophy

Animations should feel expensive.

Use:

* opacity
* translateY
* scale

Avoid:

* bounce
* spinning
* flashy effects

Duration:

200ms–400ms

# Image Rules

* Modern

---

# Card Consistency

Every card in the website must share:

* Same radius
* Same border style
* Same hover pattern
* Same spacing rhythm

No exceptions.

---

# Component Rules

Create reusable design tokens.

Do NOT hardcode colors repeatedly.

Create:

* Color tokens
* Spacing tokens
* Typography tokens
* Radius tokens

Reuse everywhere.

---

# Mobile Experience

Mobile is not an afterthought.

Requirements:

* Excellent readability
* Proper spacing
* Large touch targets
* No overflow issues

---

# Accessibility

Maintain:

* WCAG contrast
* Keyboard navigation
* Semantic HTML
* Proper heading hierarchy

---

# Success Criteria

When finished:

The website should feel like a premium $5,000–$10,000 custom portfolio website.

A business owner should immediately think:

"This developer looks professional and knows what they are doing."

The design should feel significantly more premium than a standard WordPress theme or default Tailwind template.
